Google OAuth로 본인 계정에 연결하고
구독 채널을 관심도 순으로 분석합니다.

1
키 입력
2
Google 로그인
3
데이터 수집
4
분석 완료
1단계 · API 키 및 OAuth 클라이언트 ID 입력

두 값 모두 Google Cloud Console에서 발급합니다. 입력값은 브라우저에서만 사용되며 외부로 전송되지 않습니다.

2단계 · Google 계정 연결
아직 연결되지 않았습니다.

클릭하면 Google 로그인 팝업이 열립니다.
YouTube 구독 데이터 읽기 권한(youtube.readonly)을 허용해주세요.

3단계 · 분석 시작

구독 채널 수에 따라 10초~1분 정도 소요될 수 있습니다.

설정 안내

  • 1 Google Cloud Console 접속 → 새 프로젝트 생성
  • 2 API 및 서비스 → 라이브러리 → YouTube Data API v3 활성화
  • 3 사용자 인증 정보 → API 키 만들기 → API 키 복사
  • 4 OAuth 동의 화면 설정 → 범위에 youtube.readonly 추가 → 테스트 사용자에 본인 Gmail 추가
  • 5 사용자 인증 정보 → OAuth 클라이언트 ID 만들기 → 유형: 웹 애플리케이션 → 승인된 JavaScript 출처에 현재 페이지 도메인 추가 → 클라이언트 ID 복사
